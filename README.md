# Fraud Detection Data Analysis Project

## Project Overview
This project focuses on detecting fraudulent banking transactions using data analysis and visualization techniques.

## Tools & Technologies Used
- Excel
- Power BI
- Power Query
- SQL

## Project Workflow

### 1. Data Collection
- FraudShield Banking raw dataset

### 2. Data Cleaning (Power Query/Excel)
- Removed duplicates
- Handled missing values
- Fixed data types

### 3. Data Analysis (SQL)
- Fraud vs Normal Count
- Fraud in International Transactions
- Fraud by Card Type
- Distance Impact
- High Risk Transactions

### 4. Dashboard Creation (Power BI)
Created interactive dashboard showing:
- Home page
- Overview
- Customers & Transaction analysis
- Fraud analysis
- Risk analysis

## Project Structure

- Dataset → Raw and Cleaned Data
- Excel → Excel Analysis
- SQL → SQL Queries 
- PowerBI → Dashboard Files
- Screenshots → Dashboard Images

## Key Insights
- Fraud higher in international transactions
- Distance from home increases fraud risk
- Credit cards slightly more vulnerable.
